# docs/archive/

Эта папка зарезервирована для **устаревших версий** документов проекта MedPlatform-RIS, сохранённых для истории.

## Содержимое

| Файл | Описание | Заменён на |
|---|---|---|
| `TZ_RIS_v1_archive.md` | Первая версия ТЗ (до миграции на PostgreSQL и пересмотра scope) | `backend/TZ.md` v3.0 |

## Политика

- Файлы в этой папке **не обновляются** и **не актуальны**.
- Все новые версии документов живут в `backend/`, `docs/` или `frontend/`.
- При импорте чужих документов от других проектов (например, документы прошлой группы на практике) — **не класть их сюда**: это место только для архивов MedPlatform-RIS.
